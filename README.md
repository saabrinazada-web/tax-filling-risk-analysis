# Tax Filing Risk Analysis

## Project Overview
This project presents a structured tax filing compliance risk analysis using a publicly available simulated dataset from Kaggle. The objective is to develop a statistically validated classification model to identify determinants of filing inconsistency and evaluate compliance risk patterns.

Although based on simulated data, the framework demonstrates how quantitative modeling can support risk-based compliance monitoring in corporate environments.

---

## Objectives
- Identify key predictors influencing tax filing compliance behavior  
- Develop a logistic regression-based risk classification model  
- Validate statistical robustness through diagnostic testing  
- Translate model outputs into interpretable compliance risk insights  

---

## Dataset
The dataset used in this project is a simulated structured dataset sourced from Kaggle. It is utilized solely for methodological demonstration and modeling purposes.

---

## Methodology
- Data cleaning and preprocessing (missing values handling, encoding categorical variables)  
- Exploratory data analysis (EDA)  
- Logistic regression modeling  
- Multicollinearity assessment using Variance Inflation Factor (VIF)  
- Linearity-of-logit validation  
- Odds ratio interpretation  
- Model performance evaluation  

---

## Key Results
- Developed a statistically validated compliance risk classification model  
- Identified significant predictors of filing inconsistency  
- Generated interpretable risk-oriented insights for structured compliance evaluation  

---

## Tools & Technologies
- Python (Pandas, NumPy, Statsmodels, Scikit-learn)  
- Logistic Regression  
- Statistical Diagnostics  
- Data Cleaning & Feature Engineering  

---

## Business Relevance
This framework illustrates how quantitative risk modeling can be applied to:

- Compliance monitoring systems  
- Risk-based audit prioritization  
- Early anomaly detection  
- Structured corporate tax risk assessment  

The approach emphasizes interpretable statistical modeling to support decision-making processes.
